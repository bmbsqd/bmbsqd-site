# Layouts - Bmbsqd Site

## BaseLayout.astro
- Single layout wrapping all pages
- HTML structure: head (SEO meta, fonts, OG tags) + body
- Dark/light theme support via ThemeManager (CTRL+B toggle)
- Font imports: @fontsource/lora, Stack Sans Notch/Text
- Schema.org JSON-LD for organization
- Full-width body, sections stacked vertically
