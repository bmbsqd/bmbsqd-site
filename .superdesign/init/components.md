# Components - Bmbsqd Site

## Page Sections (Astro Components)
- **HeroSection.astro** - Full-screen hero with aurora background, animated headline, dual CTAs
- **GlobalPresence.astro** - 3-column grid showing LA, Stockholm, Singapore offices
- **TrackRecord.astro** - Dark bento grid with stats ($100M+ exits, 25+ years, Fortune 500)
- **TechnologyExpertise.astro** - 3-column cards for Web3, AI/ML, Enterprise expertise
- **ServiceOfferings.astro** - Dark bento layout with 5 service categories
- **ContactSection.astro** - 2-column contact info with email, phone, offices

## UI Components (React/TSX)
- **shape-landing-hero.tsx** - HeroGeometric with Framer Motion fadeUp animations
- **aurora-background.tsx** - Animated gradient background (blue/indigo/violet)
- **client-logo-marquee.tsx** - Horizontal scrolling client names (Sony, Paramount, etc.)
- **scroll-reveal.tsx** - Scroll-triggered fade+slide animation wrapper
- **rainbow-button.tsx** - Multi-gradient animated CTA button

## Base UI (shadcn/ui style)
- Button (CVA variants: default, destructive, outline, secondary, ghost, link)
- Card (Header, Title, Description, Content, Footer, Action)
- Input, Textarea, Label, Separator, Badge, Switch
